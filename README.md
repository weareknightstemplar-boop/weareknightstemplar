## 🚀 Quick Start (Do This Now)

### Step 1: Upload Website
1. Go to **github.com** → Your repository
2. Delete old `index.html`
3. Upload NEW `index.html` + `book_cover_new.png`
4. Commit changes
5. Vercel auto-deploys in 30 seconds

### Step 2: Set Up Gumroad Products
1. Create FREE product: "7 Days of Prayer" ($0)
2. Upload: `7-Prayers-for-the-Modern-Warrior-FREE.pdf`
3. Create PAID product: "30-Day Warrior Challenge" ($27)
4. Upload: `30-Day-Warrior-Challenge-DELUXE.pdf`

### Step 3: Create Gumroad Workflow
1. Go to **Gumroad** → **Workflows**
2. Create new workflow
3. Trigger: "When someone buys 7 Days of Prayer"
4. Add 7 emails (copy from `GUMROAD_AUTOMATION_SETUP.md`)
5. Activate workflow

### Step 4: Add Patreon Link
1. Get your Patreon link
2. Update in `index.html` (search for `[YOUR_PATREON_LINK]`)
3. Re-upload to GitHub
