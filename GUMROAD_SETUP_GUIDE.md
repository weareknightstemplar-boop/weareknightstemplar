## STEP 5: UPLOAD YOUR PRODUCT FILE

1. Click "Add a file"
2. Upload the PDF version of the 30-Day Warrior Challenge
3. **File naming:** `30-Day-Warrior-Challenge.pdf`

**To create the PDF:**
- Open the `.md` file in Google Docs
- File → Download → PDF Document (.pdf)
- Or use a Markdown to PDF converter online
